A repository for benchmarks comparing the `Manycore Form Compiler`_ (MCFC) and
Fluidity_ to FFC_ and PyDOLFIN_.

Getting started
===============

Run ``make`` in the ``mesh`` directory.

.. _Fluidity: https://launchpad.net/fluidity
